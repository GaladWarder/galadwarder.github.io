The Backgrounds of BattlePlan!

While creating BattlePlan!, we wanted to stay as true as possible to Chris Wahl's iconic style, the same style that the original Inhabitants PFP project was created in. The concept was that the game should feel like the pages of a comic book coming to life, so anywhere possible we used original artwork and heavily emphasized the comic book style.

The planets themselves presented many challenges - they needed to parallax smoothly to give a sense of depth, and we needed to create heavily optimized game-ready spritesheets from high-res illustrations. Since the game loads in WebGL, every element needs to be intentional to keep load times low. The result is a set of beautiful, immersive environments that are randomly selected from for the brawls to take place in.
